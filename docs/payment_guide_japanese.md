## 🏮 ご来店前のご案内

### ご予約について
- **コース料理は事前予約制**となっております
- 一品料理のみのご利用は、ご予約なしでもご来店いただけます（空席状況によります）
- 4名様以上のグループはご予約をおすすめいたします

### お席のご案内
| 席種 | 説明 | 備考 |
|------|------|------|
| **カウンター席** | 大将の目の前でお料理を楽しめる特等席 | お一人様やカップルに最適 |
| **個室** | プライベートな空間 | 要予約・サービス料がかかる場合がございます |

### お通しについて
ご着席後、最初に小さな一品料理（**お通し**）をお出しいたします。

- 居酒屋の一般的な慣習です
- 席料を兼ねております（お一人様 ¥500〜¥800 程度）
- 当店の大将が心を込めてご用意する、ささやかなおもてなしです

## 🍽️ お食事中のご案内

### ご注文の流れ
1. まず**おしぼり**をお持ちいたします — お手をお拭きください
2. 最初にお飲み物をご注文ください — 日本酒のおすすめもお伺いいたします
3. お料理は少しずつご注文いただくのが居酒屋スタイルです
4. お料理は**皆さまでシェア**してお楽しみください
5. お食事中、いつでも追加ご注文いただけます

### 海外からのお客様への日本語ガイド

以下のフレーズをお客様にお伝えいただくと、コミュニケーションがスムーズです：

| 英語 | 日本語 | 場面 |
|------|--------|------|
| "Excuse me" | すみません | スタッフを呼ぶとき |
| "I'd like to order" | 注文お願いします | ご注文時 |
| "What do you recommend?" | おすすめは何ですか？ | おすすめを聞くとき |

## 💴 お会計について

### お会計の流れ

1. お帰りの際に**「お会計お願いします」**とスタッフにお声がけください
2. お席でのお会計、またはレジにてお会計をお願いしております
3. **伝票の内訳：**

| 項目 | 内容 |
|------|------|
| お料理・お飲み物 | ご注文いただいたすべてのメニュー |
| お通し | 席料（お一人様ごと） |
| 消費税 | 表示価格に含まれております（税込） |
| サービス料 | 個室ご利用の場合に頂戴する場合がございます |

### お支払い方法

| 方法 | 対応 | 備考 |
|------|------|------|
| 💴 **現金** | ✅ 対応 | |
| 💳 **クレジットカード** | ✅ 対応 | Visa、Mastercard など |
| 📱 **QRコード決済** | お問い合わせください | PayPay など |

### 海外のお客様へのご案内事項

インバウンドのお客様から多く寄せられるご質問・注意点です：

- **チップ不要** — 日本ではチップの習慣がありません。最高のおもてなしは標準のサービスに含まれています
- **お会計は一括** — テーブル単位での一括会計が基本です。個別会計は原則承っておりません
- **外貨のお取り扱い** — 日本円のみ対応しております
- **領収書** — レシートは自動発行。宛名入りの領収書が必要な場合はお申し付けください
