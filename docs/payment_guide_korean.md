## 🏮 식사 전 안내

### 예약
- **코스 요리는 사전 예약 필수**
- 단품 주문은 예약 없이 방문 가능 (빈자리 상황에 따름)
- 4명 이상은 예약을 권장합니다

### 좌석
| 유형 | 설명 | 비고 |
|------|------|------|
| **카운터** | 셰프 앞 좌석 | 1~2인에 적합 |
| **프라이빗 룸** | 독립된 개인실 | 예약 필수, 서비스료 별도 가능 |

### 오토시 (お通し) — 테이블 차지
착석하시면 **"오토시"**라는 작은 안주가 제공됩니다.

- 일본 이자카야의 표준 관행입니다
- 테이블 이용료 역할 (보통 1인당 ¥500~¥800)
- **거절할 수 없습니다** — 일본에서는 정상적인 문화입니다
- 작은 안주가 포함된 자릿세라고 생각하시면 됩니다

## 🍽️ 식사 중

### 주문 방법
1. 직원이 **오시보리 (물수건)**을 가져다 줍니다 — 손을 닦아주세요
2. 먼저 음료를 주문하세요 — 일본에서는 음료부터 시작합니다
3. 요리는 천천히 주문하세요 — 이자카야는 느긋하게 즐기는 곳입니다
4. 요리는 **함께 나누어** 드세요
5. 식사 중 언제든 추가 주문 가능합니다

### 유용한 일본어 표현

| 한국어 | 일본어 | 발음 |
|--------|--------|------|
| "저기요" (직원 호출) | すみません | *스미마셍* |
| "주문할게요" | 注文お願いします | *추몽 오네가이시마스* |
| "추천 메뉴가 뭔가요?" | おすすめは何ですか？ | *오스스메와 난데스카?* |
| "오마카세로 해주세요" | おまかせでお願いします | *오마카세데 오네가이시마스* |
| "맛있어요!" | おいしい！ | *오이시이!* |
| "하나 더 주세요" | もう一つお願いします | *모 히토츠 오네가이시마스* |
| "___에 알레르기가 있어요" | ___アレルギーがあります | *___ 아레루기가 아리마스* |
| "생선회는 빼주세요" | 生魚はなしでお願いします | *나마자카나와 나시데 오네가이시마스* |
| "계산서 주세요" | お会計お願いします | *오카이케이 오네가이시마스* |

### 식사 에티켓
- **젓가락**: 밥에 젓가락을 꽂아두지 마세요 (장례 의식을 연상시킵니다)
- **사케**: 서로 따라주는 것이 예의입니다
- **소리**: 면이나 국물을 먹을 때 소리를 내도 괜찮습니다!
- **신발**: 신발을 벗을 필요가 없습니다

## 💴 결제 — お会計

### 결제 방법

1. 나가실 준비가 되면: **"오카이케이 오네가이시마스"** (お会計お願いします)라고 말씀하세요
2. 직원이 계산서를 테이블로 가져오거나, 입구 쪽 카운터로 안내합니다
3. **계산서 항목:**

| 항목 | 설명 |
|------|------|
| 음식 & 음료 | 주문하신 모든 메뉴 |
| 오토시 | 테이블 차지 (인원수만큼) |
| 소비세 | 표시 가격에 이미 포함 (税込) |
| 서비스료 | 프라이빗 룸 이용 시 부과 가능 |

### 결제 수단

| 수단 | 가능 여부 | 비고 |
|------|-----------|------|
| 💴 **현금 (엔화)** | ✅ 가능 | 가장 확실한 방법 |
| 💳 **신용카드** | ✅ 가능 | Visa, Mastercard 등 |
| 📱 **QR 결제** | 문의 필요 | PayPay 등 |
| 💵 **외화** | ❌ 불가 | 일본 엔화만 사용 가능 |

> 💡 *예약 시 또는 방문 시 결제 수단을 미리 확인하시는 것을 권장합니다.*

### 일본 결제 문화 안내

- **팁 없음** — 일본에서는 팁 문화가 없습니다. 훌륭한 서비스가 기본이며, 팁을 주면 오히려 당황할 수 있습니다.
- **카운터 결제** — 많은 이자카야에서는 테이블이 아닌 입구 카운터에서 결제합니다.
- **일괄 결제** — 보통 테이블 단위로 하나의 계산서가 나옵니다. 개별 계산(더치페이)은 일반적으로 제공되지 않으니, 나중에 자체적으로 정산해 주세요.
- **영수증** — 일반 영수증(レシート)은 제공됩니다. 정식 영수증(세금계산서)이 필요하시면 **"료슈쇼 오네가이시마스"** (領収書お願いします)라고 말씀하세요.
