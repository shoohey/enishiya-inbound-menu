## 🍶 お飲み物

### 日本酒
当店自慢の品揃え。全国の蔵元から厳選した銘酒を、季節のお料理に合わせてご用意しております。

| 種類 | 特徴 |
|------|------|
| **純米大吟醸** | 華やかな香りとフルーティーな味わい — 日本酒の最高峰 |
| **純米吟醸** | 爽やかな香りとキレのある後味 |
| **純米酒** | 米の旨みがしっかり — 味の濃い料理に好相性 |
| **本醸造** | 軽やかで辛口 — 飲みやすい一杯 |

> 💡 *迷ったら「おまかせ」とお申し付けください。お料理に合わせてお選びいたします。*

**日本酒の温度帯：**
- **冷酒（れいしゅ）** — 爽やかに、香りが際立つ
- **冷や（ひや）** — 常温で味わいのバランスが整う
- **ぬる燗（ぬるかん）** — ほんのり温かく、まろやかに
- **熱燗（あつかん）** — しっかり温めて、芯から温まる
