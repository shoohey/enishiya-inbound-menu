お通しについて — About "Otōshi"

ご着席後、最初に小さな一品料理をお出しいたします。これは居酒屋の一般的な慣習で、席料を兼ねたものです（お一人様 ¥500〜¥800 程度）。大将が心を込めてご用意する、ささやかなおもてなしです。

When seated, a small appetizer called "Otōshi" will be served. This is a standard Japanese izakaya custom that serves as a table charge (approx. ¥500–¥800 per person). It is not optional.

About "Otōshi" — お通し

When seated, you'll receive a small appetizer. This is a standard izakaya custom that serves as a table charge (¥500–¥800 per person). It is not optional — think of it as a cover charge with a complimentary dish.

How to Order
Ordering Flow
  1. A hot towel (Oshibori) will be brought — clean your hands
  2. Start with drinks — try saying "Toriaezu, biiru!" (Beer first!)
  3. Order dishes gradually — izakaya dining is leisurely
  4. Dishes are meant to be shared among the table
  5. Order more anytime throughout the evening

入座后服务员会送上一道小菜。这是日本居酒屋的标准惯例，相当于席位费（每人约 ¥500–¥800）。不可拒绝 — 这是日本的正常消费习惯。
